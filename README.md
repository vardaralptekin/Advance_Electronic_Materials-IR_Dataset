# Advance_Electronic_Materials-IR_Dateset
Low-resolution thermal image dataset for people counting and indoor positioning, recorded using a Heimann HTPA32x32 thermopile array. Designed for privacy-preserving perception, neural model compression, and hardware-aware inference research. Submitted to Advanced Electronic Materials
